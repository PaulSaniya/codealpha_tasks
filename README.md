# CodeAlpha Tasks

This repository contains the projects completed as part of the CodeAlpha Internship Program.

## Projects Included

### 1. Ecommerce Web Application

A full-stack Ecommerce Web Application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application allows users to browse products, view product details, add items to the cart, and place orders.

#### Features
- User Registration and Login with JWT Authentication
- Product Listing and Detailed Product Pages
- Shopping Cart Functionality
- Order Placement
- Image Upload Support
- Responsive User Interface

#### Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer / Cloudinary

---

### 2. Social Media Mobile Application

A React Native based Social Media Application that allows users to register, log in, create posts, upload images, and interact with other users through likes and comments.

#### Features
- User Registration and Login
- Create Text and Image Posts
- News Feed Display
- Like and Comment Functionality
- User Profile Screen
- Image Upload Support
- Responsive Mobile UI

#### Technologies Used
- React Native
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer / Cloudinary
